# Travel Agent AI

An AI-powered travel planning application that generates personalized itineraries using live flight and hotel data.

## Overview

Users enter a destination, travel dates, and budget. The app returns a day-by-day itinerary with flight options, hotel recommendations, and activity suggestions — all generated by an LLM with real data context.

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Backend:** Next.js API routes
- **AI:** Anthropic Claude for itinerary generation
- **Data:** Live flight and hotel APIs

## Project Structure

```
travel-agent/
├── app/
│   ├── page.tsx           # Search — destination, dates, budget
│   ├── plan/page.tsx      # Itinerary results
│   └── api/plan/route.ts  # POST handler — generates itinerary

└── lib/
```
